Syntax Errors

Answer
(1) The program must be compiled into bytecode with the Java compiler, then (2) run using the Java interpreter.

It is likely that you will do something wrong. Here is the sample program with a deliberate error: 

Class Hello
{
  public static void main ( String[] args )
  {
    System.out.println("Hello World!");
  }
}
