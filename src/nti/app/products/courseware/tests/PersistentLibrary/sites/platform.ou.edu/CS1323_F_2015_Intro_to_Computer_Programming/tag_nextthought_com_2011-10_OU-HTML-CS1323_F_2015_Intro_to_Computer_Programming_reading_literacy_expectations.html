Computer Literacy Expectations

CS 1323, Fall 2015

Students enrolled in CS 1323 are expected to be able to perform the following operations.

  1. Install software on their laptop computer.

  2. Create, rename and remove files and folders.

  3. Use file extensions to determine the type of a file.

  4. Read PDF documents using Acrobat Reader or other appropriate software.

  5. Retrieve and store information in ZIP files.

  6. Combine several files into a single ZIP file.

  7. Navigate folder structures.

  8. Locate web based materials given a URL.

  9. Use Janux to locate learning materials and submit work.

  10. Read and send electronic mail through the standard university account.

  11. Share directories and files using Dropbox.

  12. Create documents using Microsoft Word, including creating figures and diagrams within the document.

  13. Convert Word documents to PDF format.

  14. Use simple text based spreadsheets in Excel.
